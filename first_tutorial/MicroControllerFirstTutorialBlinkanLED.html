Microcontroller First Tutorial -  Blink and LED

This tutorial outlines how to :

  1. Setup a working development environment on a personal computer or Raspberry PI
  2. Wire up your experimental setup on a solderless prototype board or use direct wiring
  3. Connect the pc or RPI to the microcontroller
  4. Compile software and flash compiled code onto the microntroller
  5. Run the software and view console
  6. Change code Recompile and run again

What you will need for this tutorial:

  1. A laptop or a rpi working as a desktop  with connection to the internet (WIFI or ethernet)
  2. 1 esp8266 nodemcu or other arduino equivalent
  3. Solderless Protype board
  4. 2 Wires
  5. 1 USB to usb mini connector for connecting to arduino
